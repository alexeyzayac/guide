### 01. Настройте репозиторий `apt` Docker
```bash
sudo apt update
sudo apt install ca-certificates curl
sudo install -m 0755 -d /etc/apt/keyrings
sudo curl -fsSL https://download.docker.com/linux/debian/gpg -o /etc/apt/keyrings/docker.asc
sudo chmod a+r /etc/apt/keyrings/docker.asc

sudo tee /etc/apt/sources.list.d/docker.sources <<EOF
Types: deb
URIs: https://download.docker.com/linux/debian
Suites: $(. /etc/os-release && echo "$VERSION_CODENAME")
Components: stable
Architectures: $(dpkg --print-architecture)
Signed-By: /etc/apt/keyrings/docker.asc
EOF

sudo apt update
```
---
### 02. Установка пакетов Docker
```bash
sudo apt install -y \
	docker-ce \
	docker-ce-cli \
	containerd.io \
	docker-buildx-plugin \
	docker-compose-plugin
```
---
### 03. Проверка и запуск 
```bash
sudo systemctl status docker
sudo systemctl start docker
```
---
### 04. Настройка прав Docker для текущего пользователя
```bash
#создаём группу 'docker' (если она ещё не существует)
sudo groupadd docker

#добавляем текущего пользователя в группу 'docker'
sudo usermod -aG docker $USER

#активируем изменения группы в текущей сессии (без перезагрузки)
newgrp docker
```
Настройка автоматического запуска Docker и containerd:
```bash
sudo systemctl enable docker.service
sudo systemctl enable containerd.service
```
