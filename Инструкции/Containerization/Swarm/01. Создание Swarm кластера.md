			Чтобы кластер работал корректно через публичные IP, на всех ВМ должны быть открыты следующие порты (в облачном firewall / security group):
- **2377/tcp** – для управления Swarm (только между менеджером и воркерами)
- **7946/tcp и 7946/udp** – для связи между узлами (gossip)
- **4789/udp** – для overlay-сетей (VXLAN)
### Создание Swarm кластера (1 мастер, 2 ноды)
#### 1. На машине менеджере 
Инициализируем Swarm и указываем advertise-адрес (публичный IP менеджера, чтобы воркеры могли подключиться):
```bash
docker swarm init --advertise-addr <IP менеджера>
```
После успешной инициализации вы увидите команду для добавления рабочих нод.

---
#### 2. На машинах воркерах 1 и 2 выполните команду из пункта выше
Это подключит ноду к машине менеджеру (пример ниже):
```bash
docker swarm join --token SWMTKN-1-... <IP менеджера>:2377
```
---
#### 3. Проверка списка нод
На **машине менеджера** выполните:
```bash
docker node ls
```
Обратите внимание:
- Менеджер имеет статус `Leader` (если он единственный менеджер).
- Воркеры не имеют `MANAGER STATUS`.
- Все ноды должны быть `Ready` и `Active`.
---
### Как посмотреть токены
Все команды нужно выполнять на **главной manager-ноде**:
```bash
docker swarm join-token worker #посмотреть токен для worker-нод
docker swarm join-token manager #посмотреть токен для manager-нод
```
**Получить только сам токен**:
```bash
docker swarm join-token -q worker
docker swarm join-token -q manager
```
#### Новая генерация токена
```bash
docker swarm join-token --rotate worker
docker swarm join-token --rotate manager
```

### Управление узлами
Управление режимом доступности узла 
```bash
#перевести менеджера в режим «только управление»
docker node update --availability drain <имя менеджера>

#вернуть менеджеру режим управления
docker node update --availability active <имя менеджера>

docker node update --availability pause <узел>
```
Повышение и понижение роли узла (менеджер ↔ воркер)
```bash
#повысить воркера до менеджера
docker node promote <имя воркера>

#понизить менеджера до воркера
docker node demote <имя менеджера>
```
