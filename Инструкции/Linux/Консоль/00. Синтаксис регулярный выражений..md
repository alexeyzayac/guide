
| Cответствие                                                    | Используемые выражения | Пример                                  |
| -------------------------------------------------------------- | ---------------------- | --------------------------------------- |
| Начало строки                                                  | ^                      | ^a → строка начинается с а              |
| Конец строки                                                   | $                      | ?$ → строка заканчивается ?             |
| Любой одиночный символ (кроме новой строки)                    | .                      | al..ey → alexey, al -ey, al tey         |
| Экранирование: убирает специальное значение следующего символа | \                      | \$asa → $asa                            |
| 0 или более повторений предыдущего элемента                    | *                      | ab*c → ac, abc, abbc                    |
| Любой одиночный символ в наборе                                | [characters]           | [Na] → Nikita (любая гласная)           |
| Любой одиночный символ не входящий в наборл                    | [^ characters]         | [^ teAr] → Artem<br>[^ Atrem] → пустота |
| Любой символ в диапазоне с1-с2                                 | [c1-c2]                | [1-9] → любая цифра                     |
| Любой символ вне диапазоне с1-с2                               | [^ c1-c2]              | [1-9] → любой не цифровой символ        |
| Буква (предопределённый POSIX-класс)                           | `[[:alpha:]]`          | `[[:alpha:]]` → любая буква             |
| Цифра                                                          | `[[:digit:]]`          | `[[:digit:]]` → `0`-`9`                 |
| Буква или цифра                                                | `[[:alnum:]]`          | `[[:alnum:]]` → `0`-`9` `a`-`z`         |
| Пробельные символы (пробел, табуляция и др.)                   | `[[:space:]]`          | `[[:space:]]` → ` `                     |
| Знаки пунктуации                                               | `[[:punct:]]`          | `[[:punct:]]` → `.,!?;:` и т.п.         |
| Заглавные буквы                                                | `[[:upper]]`           | `[[:upper]]`→ ABC                       |
| Не заглавные буквы                                             | `[[:lower:]]`          | `[[:lower:]]`→ abc                      |
| Любое из двух выражений                                        | 'e1\\\|e2'             | 'one\\\|two' → one или two              |
