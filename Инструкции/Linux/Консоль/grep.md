# Команда grep
## Базовые команды
```bash
grep Egor prm.txt #поиск в файле

grep -w egor prm.txt #искать точечное соответствие

grep -i Egor prm.txt #игнорировать регист

grep -l Egor * #в каком файле находится соответствие
```
## Регулярные выражения
```bash
#поиск всеъ строк начинающихся с заглавной буквы
grep '^[A-Z]' prm.txt

#найти пустые строки
grep '^$' prm.txt

#найти не пустые строки
grep -v '^$' prm.txt

#найти строки либо содержашие либо Egor либо Artem
grep 'Egor\|Artem' prm.txt

#найти строки содержашие минимум 6 символов
grep '......' prm.txt

#найти все строки где символ меньше появляется перед символом больше
grep '<.*>'

#найти все строки где после w идёт .
grep 'w\.' prm.txt

#если нужно экранирровать много символов
grep -F w. prm.txt #либо использовать: fgrep w. prm.txt

#сверка отсортированного первого файла с содержимым второго файла учитывя экранированные символы 
cut -d: -f7 /etc/passwd | sort -u | grep -f /etc/shells -F
cut -d: -f7 /etc/passwd | sort -u | fgrep -f /etc/shells
```
