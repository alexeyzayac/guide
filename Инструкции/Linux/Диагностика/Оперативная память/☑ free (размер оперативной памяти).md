Утилита `free` — это простой и незаменимый инструмент для мониторинга оперативной памяти (RAM) и раздела подкачки (swap) в Linux. Она читает информацию из виртуальной файловой системы `/proc/meminfo` и представляет её в удобном табличном вида.
### Основные поля вывода
| Поле           | Описание (откуда берётся)                                                                                                                                                                                                                         |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **total**      | Общий объем доступной памяти (физической + swap).                                                                                                                                                                                                 |
| **used**       | Используемая память. Рассчитывается по формуле: `total - available`.                                                                                                                                                                              |
| **free**       | Полностью неиспользуемая память.                                                                                                                                                                                                                  |
| **shared**     | Память, используемая преимущественно `tmpfs` (временной файловой системой).                                                                                                                                                                       |
| **buff/cache** | Память, занятая под **буферы** (buffers) и **кэш** (cache) ядра. Она служит для ускорения работы системы.                                                                                                                                         |
| **available**  | **Самая важная колонка для оценки!** Показывает приблизительный объем памяти, доступный для запуска новых приложений без использования swap. В отличие от колонки `free`, она учитывает, что часть кэша может быть освобождена при необходимости. |
