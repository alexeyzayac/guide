# Команда `du` – анализ занятого места
**`du`** (от *disk usage*) – стандартная утилита Linux/Unix для оценки дискового пространства, занимаемого файлами и каталогами. В отличие от `df` (свободное место на разделе), `du` показывает, сколько места **фактически** используют выбранные объекты. Идеальна для поиска «пожирателей» места.

## ⚙️ Популярные команды
```bash
du -sh /home/user       # общий размер домашней папки
du -h --max-depth=1     # размеры подкаталогов первого уровня
du -ah | sort -h        # все файлы + сортировка по размеру
du -shc /var /tmp       # размеры нескольких каталогов + итог
```
## 📖 Полезные опции
|Опция|Длинная форма|Описание|
|---|---|---|
|`-h`|`--human-readable`|Размеры в удобном формате (К, М, Г)|
|`-s`|`--summarize`|Только общий итог для каждого аргумента (без подкаталогов)|
|`-c`|`--total`|Добавить итоговую строку в конце|
|`-a`|`--all`|Показывать размеры **всех** файлов, а не только каталогов|
|`-S`|`--separate-dirs`|Для каталогов – исключить размер подкаталогов (только собственные файлы)|
|`-d`|`--max-depth=N`|Глубина обхода не более N уровней (например `-d 1`)|
|`-x`|`--one-file-system`|Не переходить на другие ФС (полезно при замонтированных дисках)|
|`-l`|`--count-links`|Учитывать жёсткие ссылки несколько раз (по умолчанию – один раз)|
|`--exclude=ШАБЛОН`|–|Исключить файлы/каталоги, подходящие под шаблон|
|`-X`|`--exclude-from=ФАЙЛ`|Исключить шаблоны из файла (по одному на строку)|
|`--time`|–|Показать время последней модификации|
|`-B`|`--block-size=SIZE`|Задать размер блока (например `-BM` для мегабайт)|
## 🧪 Примеры команд с пояснениями
```bash
# 1. Общий размер каталога /home/user
du -sh /home/user

# 2. Найти 5 самых больших подкаталогов в корне (игнорируя ошибки доступа)
du -h --max-depth=1 / 2>/dev/null | sort -hr | head -n 5

# 3. Анализ текущего каталога: размеры всех элементов первого уровня, отсортированные
du -ah --max-depth=1 | sort -h

# 4. Размеры нескольких папок и общая сумма
du -shc /home /var /opt

# 5. Показать только размеры непосредственных подкаталогов (без углубления)
du -h --max-depth=1

# 6. Исключить все файлы .log и временные файлы
du -sh --exclude="*.log" --exclude="*.tmp" /var/log

# 7. Рекурсивно с глубиной 2, только человеко-читаемый
du -h --max-depth=2 /opt
```
