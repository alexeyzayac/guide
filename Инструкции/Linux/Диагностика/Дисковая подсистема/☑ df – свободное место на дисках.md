# Команда `df` – свободное место на дисках

**`df`** (от *disk free*) – стандартная утилита Linux/Unix для отображения использования дискового пространства на смонтированных файловых системах. Позволяет быстро оценить, сколько места занято и сколько осталось.
## ⚙️ Популярные команды
```bash
df -h               # человеко-читаемый вид (КБ, МБ, ГБ)
df -h /home         # только ФС, содержащая /home
df -hT              # добавить тип файловой системы (ext4, xfs, tmpfs и т.д.)
df -i               # показать использование inode (а не блоков)
df --total -h       # добавить итоговую строку в конце
```
## 📖 Полезные опции

| Опция     | Длинная форма        | Описание                                                        |
| --------- | -------------------- | --------------------------------------------------------------- |
| `-h`      | `--human-readable`   | Размеры в **1024** байтах (К, М, Г)                             |
| `-H`      | `--si`               | Размеры в **1000** байтах (КБ, МБ, ГБ)                          |
| `-T`      | `--print-type`       | Показывает тип файловой системы                                 |
| `-i`      | `--inodes`           | Количество использованных и свободных inode                     |
| `-t`      | `--type=тип`         | Показать только ФС указанного типа (например, `-t ext4`)        |
| `-x`      | `--exclude-type=тип` | Исключить ФС указанного типа  (например, `-x tmpfs`)            |
| `-l`      | `--local`            | Только локальные ФС (без сетевых, например NFS)                 |
| `-a`      | `--all`              | Включая виртуальные и псевдо-ФС (proc, tmpfs, udev)             |
| `--total` | –                    | Добавить итоговую строку с общими значениями                    |
| `-B`      | `--block-size=SIZE`  | Установить размер блока для вывода (например `-BM` – мегабайты) |
## 🧪 Примеры команд с пояснениями
```bash
# 1. Самый частый запрос – свободное место в /home
df -h /home

# 2. Все ФС, включая виртуальные (например, tmpfs)
df -ah

# 3. Тип ФС + читаемые размеры (полезно для проверки ext4/xfs)
df -hT

# 4. Inode – когда места много, а новые файлы создать нельзя
df -i /var/log

# 5. Итог только по дискам ext4 (отбросить tmpfs, devtmpfs)
df -ht ext4 --total

# 6. То же самое, но итоговая строка одна (уберем заголовки и прочее)
df -Ht ext4 --total | tail -n 1
```
## 🔗 Связанные команды
- `lsblk` – список блочных устройств (дисков, разделов)
- `mount` – просмотр смонтированных ФС
- `du` – оценка места, занятого файлами/каталогами
- `find` – поиск больших файлов
- `ncdu` – интерактивный анализатор дискового пространства
