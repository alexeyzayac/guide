Команда `df` (disk free) — это стандартная утилита в Linux и Unix-системах, которая показывает информацию об использовании дискового пространства на всех примонтированных (подключённых) файловых системах. Это основной инструмент для быстрой проверки свободного места на дисках.
```bash
df [ОПЦИИ] [ФАЙЛ/КАТАЛОГ]
```
#### Полезные опции:
**-h** - Выводит размеры в читаемом формате
**-H** - Аналогична `-h`, но использует степени 1000 (1 MB = 1000 KB) вместо 1024
**-t** - Выводит информацию **только** для файловых систем указанного типа (df -t ext4)
-T - Добавляет в вывод колонку с типом файловой системы
**-i** - Вместо дискового пространства показывает информацию об использовании индексных дескрипторов (inode)
**-l** - Показывает только локальные файловые системы, исключая сетевые
**-a** - Показывает **все** файловые системы, включая виртуальные и псевдо-ФС
**--total** - Добавляет в конец вывода итоговую строку с суммой по всем отображённым файловым системам
#### Примеры команд:
```bash
#покажет, сколько свободного места осталось на файловой системе, в которой находится папка
df -h /home

#просмотр всех ФС (включая виртуальные):
df -ah

#покажет использование inode на файловой системе, в которой находится папка
df -i /home

#общий итог по дискам типа ext4ыы
df -Ht ext4 --total | tail -n 1
```
