`iotop` — это интерактивная утилита для мониторинга дисковой активности в Linux. Она работает по принципу `top`, но вместо загрузки процессора и памяти показывает в реальном времени, какие процессы и насколько интенсивно читают или пишут данные на диск.
#### Полезные опции:
**-b** - Включить пакетный (неинтерактивный) режим для записи в файл
**-d** - Установить задержку между обновлениями в секундах
**-o** - Показать только процессы, которые **реально** выполняют I/O
**-a** - Показывать накопленный объем данных вместо текущей скорости
**-q** - Уменьшить количество выводимой информации
**-t** - добавить временную метку к каждой строке вывода
**-u USER** - Показать активность только для указанного пользователя 
**-P** - показать только процессы .
#### Примеры команд:
```bash
#показать только активные процессы
iotop -o

#изменение интервала обновления
iotop -d 2

#отслеживать конкретный процесс
iotop -p 1234

#мониторинг активности конкретного пользователя
iotop -u username

#пакетный режим с временными метками
iotop -botq -d 2 >> disk_war.log
cat disk_war.log | grep -v "0.00 B/s" # Убираем строки с нулевой активностью
```

## Как читать вывод `iotop`
### Стандартный вывод
Верхняя часть экрана показывает общую статистику:
- **Total DISK READ / WRITE**: Общая пропускная способность чтения/записи между процессами и подсистемой ядра.
- **Current DISK READ / WRITE**: Фактическая скорость чтения/записи на уровне физического устройства.

Основная таблица содержит следующие столбцы:
- **TID**: ID потока (или PID процесса).
- **PRIO**: Приоритет ввода/вывода процесса.
- **USER**: Пользователь, запустивший процесс.
- **DISK READ / WRITE**: Скорость чтения/записи процессом за текущий интервал измерения.
- **SWAPIN**: Процент времени, которое поток потратил на подкачку из swap.
- **IO**: Процент времени, которое поток провел в ожидании операций ввода/вывода.
- **COMMAND**: Имя команды процесса.

### Интерактивные клавиши

Во время работы `iotop` можно управлять отображением с помощью горячих клавиш. Вот самые полезные из них:

- **Клавиши со стрелками влево/вправо**: Меняют столбец, по которому происходит сортировка.
- **r**: Изменяет порядок сортировки на обратный.
- **o**: Показывает только процессы, у которых есть активность ввода/вывода (аналог опции `--only`).
- **p**: Переключает режим отображения: показывать все потоки или только процессы.
- **a**: Переключает режим отображения: показывать текущую скорость или накопленный объем данных.
- **q**: Выход из программы.
